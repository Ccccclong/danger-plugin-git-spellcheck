> This plugin checks the spelling in code and reports any error as markdown PR comment.

## Usage

Install:

```sh
yarn add danger-plugin-git-spellcheck --dev
```

At a glance:

```js
// dangerfile.js
import gitSpellcheck from 'danger-plugin-git-spellcheck'

gitSpellcheck()
```
